# Onboarding
Herzlich willkommen auf der ICT-Webseite unserer Schule! Hier finden Sie Antworten und Lösungen bei technischen Fragen und Problemen. Speichern Sie diese Seite deshalb am besten gleich in Ihren Favoriten / Lesezeichen ab oder notieren Sie sich die Adresse [ict.gbsl.website](https://ict.gbsl.website) für später.

Arbeiten Sie den aktuellen Artikel durch, um Ihren Laptop und Ihr Smartphone für den Gebrauch an unserer Schule einzurichten. **Die meisten Einrichtungsschritte können Sie bereits jetzt zu Hause erledigen, damit Sie für den ersten Schultag bereit sind**.

Falls Sie beim Einrichten Hilfe benötigen, können Sie sich beim IT-Support für Schüler:innen melden: <TLink id="support.students.email" />. Am besten schreiben Sie sich diese E-Mail-Adresse ebenfalls irgendwo auf, wo Sie sie jederzeit wiederfinden.

:::danger[Altes Office-Konto entfernen]
Falls Sie von ihrer alten Schule ein Office365-Konto haben, müssen Sie dieses zuerst entfernen, bevor Sie das neue GBSL-Konto einrichten können. Befolgen Sie dazu [diese Anleitung](#TODO).
:::

## Einrichtung vor dem ersten Schultag
Den Grossteil der Geräteeinrichtung können Sie bereits jetzt zu Hause erledigen. Es wird erwartet, dass Sie die Schritte in diesem Abschnitt **bis am 15. Juni vollständig erledigt haben**. 

Melden Sie sich bei Problemen rechtzeitig beim <TLink id="support.students.email">IT-Support für Schüler:innen</TLink>. Weitere Unterstützung erhalten Sie auch in der [Anleitung von EDUBERN](https://erzbe.sharepoint.com/sites/EDUBERN-Infohub-Hilfsmittel/Lists/Hilfsmittel/Attachments/157/Anleitung%20–%20Onboarding%20EDUBERN%20BYOD%20FED_RESET_D.pdf) (dem Informatikdienstleister unserer Schule), sowie von deren [Chatbot](https://qr.edubern.ch/assistent).
