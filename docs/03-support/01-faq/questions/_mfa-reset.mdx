<Faq>
    #### Mein Office 365 __Schulkonto ist zurückgesetzt__ worden. Was muss ich nun tun?
    <Solution>
        Neukonfiguration MFA:

        <Steps>
            1. Navigieren Sie zu https://qr.edubern.ch/mfasetup
            2. Mit der Schul-E-Mail-Adresse und Passwort anmelden.  
               Es erscheint die Meldung: «Weitere Informationen erforderlich». Auf __Weiter__ klicken.
               Bei Fenster «Angemeldet bleiben?» auf __Ja__ klicken.
               Bei Fenster «Microsoft Authenticator» unten links auf __Ich möchte eine andere Methode einrichten__ klicken.
            3. Methode «Telefon» gemäss Anweisungen auf Bildschirm einrichten.
            4. Erneute Navigation zu https://qr.edubern.ch/mfasetup
        </Steps>
        Im neuen Fenster __Anmeldemethode hinzufügen__ auswählen.
        Methode «Microsoft Authenticator» gemäss Anweisungen auf dem Bildschirm einrichten.
        Anschliessend kann die MFA für das Login wieder wie gewohnt verwendet werden. Weitere Informationen und Anleitungen dazu sind auf der [Website von EDUBERN](https://www.edubern.bkd.be.ch/de/start/portal.html) zu finden.
    </Solution>
</Faq>
